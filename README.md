 Project Overview
This repository contains the complete asset production phase of a final-year software engineering capstone project. Developed intensively over the course of one academic semester, this project focuses on the intersection of complex 3D geometric modeling, spatial environment layouts, and engine-optimized asset generation. 

The primary deliverable is an optimized, high-fidelity scene tailored for seamless integration with interactive real-time runtime engines.
